# BreadQuest

**BreadQuest** é um jogo de aventura e escalada onde o jogador controla um personagem que, ao sair para comprar pão, precisa enfrentar uma montanha repleta de obstáculos e perigos. Com habilidade e determinação, o jogador deve guiar o personagem até o topo para alcançar o tão desejado pão e retornar em segurança!

## 🎯 Objetivo do Jogo

O objetivo em **BreadQuest** é conduzir o personagem por uma série de desafios em uma montanha traiçoeira. Se o jogador falhar ao evitar um obstáculo, ele volta ao ponto de partida, necessitando habilidade e paciência para alcançar o topo.

## ✨ Funcionalidades

- Obstáculos variados e fixos em diferentes pontos da montanha
- Controles simples: pule, desvie e entre em passagens
- Sistema de uma vida: cada erro reinicia a jornada

## 🚀 Instalação

Para rodar **BreadQuest** no seu ambiente local:

1. Clone este repositório:
  ```bash 
   git clone https://github.com/seu-usuario/BreadQuest.git
```
2. Acesse a pasta do projeto:
  ```bash
   cd BreadQuest
  ```
